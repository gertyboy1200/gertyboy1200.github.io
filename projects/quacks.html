← Back to Portfolio

Project Overview

A Python implementation of the board game The Quacks of Quedlinburg. Players take turns brewing potions by drawing ingredients from a bag, aiming for high scores while avoiding explosions.

Key Features

  • Randomized bag-draw mechanics
  • Dynamic score tracking
  • Turn-based player actions

Tech Used

Python 3, Object-Oriented Programming, basic file I/O for saving game state.
